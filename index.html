Project 4: Interactive Photo Gallery

This is the first project that was focused on JavaScript. Students had to create a photo gallery that had a search/filter function and display a lightbox when images were clicked. For this project I chose to use the popular lightbox2 plugin.

Project 6: SVG Site Update

I really enjoyed this project. We had to take an existing site and change all the icons from PNG to SVG format. The best part about this project was learning about CSS animations and transitions to add some life to icons.

Project 7: Video Player

Another challenging project focused on JavaScript. Students had to build a custom video player using JavaScript. This included a custom progress bar, a time elapsed display, and full screen controls. Finally students needed to highlight the text on the page to match the video.

Skills:JavaScript, Event Listeners, HTML 5 Video

Project 9: Web Dashboard

Project 10: Using APIs

This was my favourite project of the entire Techdegree. For this project students had to use a public API to create a gallery of at least 6 items from the API. For this project I chose to use the Spotify API to display the top Kanye West tracks in Canada. This project used a combination of jQuery, AJAX, and JavaScript.

Skills: AJAX, JavaScript

View Github Page
Project 11: Performance Optimization

For this project students had to seriously optimize the performance of a photo site. The page would take 20 seconds to load on a good connection. The goal was to bring the total download size from 11MB down to 1.3MB and the number of HTTP requests down from 43 to less than 27 requests.

Skills: Chrome Developer Tools, Gulp, Script Optimization
